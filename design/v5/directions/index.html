Bloomberg для Зоны

Максимальная плотность данных: железная сетка на 1px-щелях, компартменты вместо карточек, радиусы 2px, без теней. Амбер — сигнальный цвет, JetBrains Mono для всех цифр. Лог-шкала графиков, точки-«аномалии» ниже медианы подсвечены зелёным.
