Пользователь указывает, какие размеры ему интересны для рекомендаций. ⚠️ **Проверить перед запуском фронтенда.**

## UX / Интерфейс фронтенда

### Непонятные элементы — нужно переработать

- **ID товара** — убрать из каталога и везде в интерфейсе, показывать только **название** товара
- **Категория товара** — переводить на язык интерфейса. Сейчас в БД хранится на английском (`weapon`, `armor` и т.д.), нужна локализация под выбранный язык пользователя
- **Watchlist → "Избранное"** — переименовать в интерфейсе. "Watchlist" непонятен русскоязычному пользователю
- **Поиск лотов** — должен работать по названию товара, не по ID
- **Кнопка "Обновить данные"** — показывать таймер обратного отсчёта (2 мин кулдаун), а не просто блокировать кнопку. Пользователь должен видеть "Обновить через 1:43". Вручную запускать не нужно — данные обновляются автоматически каждые 5 мин.

## Дизайн и подсказки (Tooltips)

### Общий принцип
Везде где есть непонятный термин или цифра — добавить подсказку (tooltip при наведении или иконка "?").

### Конкретные подсказки

| Элемент | Подсказка |
|---------|-----------|
| **Быстро** | "Цена чуть ниже минимума на рынке. Скорее всего продастся быстро, но заработаешь меньше." |
| **Нормально** | "Рыночная цена. Баланс между скоростью и доходом." |
| **Выгодно** | "Цена выше медианы. Придётся подождать, но заработаешь больше." |
| **~3 часа / ~18 часов** | "Прогноз: сколько времени обычно висит лот по такой цене до выкупа." |
| **Low / Стабильный** | "Цена этого товара слабо колеблется — риск купить по невыгодной цене минимален." |
| **High / Высокий риск** | "Цена сильно скачет. Сегодня выгодно, завтра — уже нет." |
| **Лучшее время** | "В этот час и день недели обычно больше всего покупок на аукционе." |
| **Уверенность (confidence)** | "Насколько точен прогноз. Зависит от количества данных о продажах." |
| **Комиссия 5%** | "Аукцион берёт 5% с каждой продажи. Уже учтено в расчёте прибыли." |
- **"3 часа", "18 часов"** — непонятно без контекста. Нужно пояснение: это *прогноз времени до продажи при выбранной цене*
- **"Быстро / Нормально / Выгодно"** — неочевидно что это значит. Добавить подсказку: *"Быстро = цена чуть ниже рынка, продастся скорее всего за ~3ч"*
- **"Low / Medium / High"** (риск) — непонятно для пользователя. Заменить на человекочитаемое: *"Стабильный товар / Умеренный риск / Высокий риск"* + короткое пояснение что это волатильность цены
- **"Лучшее время"** — неочевидно что это такое. Добавить расшифровку: *"В этот час/день обычно больше всего покупок — выгоднее выставить лот заранее"*
